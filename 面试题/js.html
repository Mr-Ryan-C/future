        # javaScript

        # 1.eval 是做什么的？

        • 它的功能是把对应的字符串解析成 JS 代码并运行

        • 应该避免使用 eval，不安全，非常耗性能（2次，一次解析成js语句，一次执行）

        # 2.什么是 window对象？ 什么是docume对象

        • window对象是指浏览器打开的窗口

        • documen对象，是HTML文档对象的一个只读引用，window对象的一个属性

        # 3.null 和 undef的区别

        • null 表示一个对象是没有值的值，也就是值为空；

        • undef 表示一个变量声明了，没有初始化赋值

        • undefeated 不是一个有效的 JSON， 而null是

        • undefeated的类型是 undefeated·

        • null的类型是 Object
